🖩 Calculette Soroban — mode d'emploi

Cette calculette utilise un boulier soroban comme interface de saisie. La logique est donc différente d'une calculette classique.

Principe de base

Tu poses une valeur sur le boulier, puis tu appuies sur ✓ pour l'enregistrer. Le boulier ne se remet pas à zéro automatiquement — tu continues de poser la valeur suivante directement.

Addition et soustraction (mode auto)

  • Pose la première valeur → ✓
  • Modifie le boulier pour obtenir la valeur suivante → ✓
  • La bande enregistre automatiquement la différence entre deux ✓ successifs (+ si augmentation, − si diminution).

Multiplication et division

  • Sélectionne × ou ÷ avant d'appuyer sur ✓
  • Pose la valeur (multiplicateur ou diviseur) sur le boulier → ✓
  • La bande enregistre l'opération explicite.

Autres boutons

  • ↺ — Annule le dernier ✓ (remet le boulier à l'état du coup précédent)
  • Résultat — Affiche la bande de calcul avec le total
  • RAZ — Remet le boulier et la bande à zéro (confirmation demandée si bande non vide)
  • 👁 / 🙈 — Masque ou affiche la valeur numérique
